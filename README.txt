本软件采用Python 3.6语言开发，UI界面是基于PYQT5框架编写的，生成的exe文件在dist文件夹中，
由pyinstaller打包生成，文件夹中已经包含有程序运行依赖的python包和图片资源，双击main.exe即可开始火柴游戏之旅。

注意自定义模式输入符合要求的算式，否则将无法运行出正确结果：
（数字 运算符 数字 运算符 数字）的形式：数字可为两位数或个位数，运算符可为加、减、乘、等号
扩展模式界面输入等式长度和形式较自由（注意不要连续输入运算符）。
×号请用叉号代替

程序运行时请关闭360等类型的杀毒软件！！

若分辨率不合适请根据需要调整电脑的分辨率。

源代码编译依赖PYQT5,math,os,sys,random,re等包，若编译源码请按需要安装。

MatchGame为工程文件夹，其中data.py为获取数据文件，search.py为搜索算法文件，
search_complex也是同样的搜索算法文件，仅仅是判断等式成立时调用了不同的函数（长等式判断使用正则表达式方法）

main.py为主界面文件，judge_equality为判断等式、转化列表等功能文件，complex_judge为判断长等式是否成立的文件

num_2.py、complex.py和help.py为ui文件生成得到，图片均为程序运行依赖的资源。
